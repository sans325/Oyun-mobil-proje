## 🚀 Nasıl Kullanılır?

### Gereken Dosyalar
```
flappy-bird/
├── advanced.html          (Ana HTML dosyası)
└── game-advanced.js       (Oyun JavaScript kodu)
```

### Kurulum (3 Adım)

1. **Dosyaları indir**
   - `advanced.html`
   - `game-advanced.js`
   - Aynı klasöre koy

2. **Tarayıcıda aç**
   - `advanced.html` dosyasını çift tıkla
   - VEYA sağ tıkla → "Tarayıcıda Aç"

3. **Oyna!**
   - SPACE tuşu veya tıkla = Zıpla
   - P tuşu = Duraklat
   - Ayarları sağ panelden düzenle

### Web Sunucusunda Yayınlama

**GitHub Pages (Ücretsiz)**
1. GitHub'da yeni repository oluştur
2. `advanced.html` ve `game-advanced.js` yükle
3. Settings → Pages → main branch seç
4. Oyunu oynat! 🚀
